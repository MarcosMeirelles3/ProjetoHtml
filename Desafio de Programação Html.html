História

Os primeiros computadores foram construídos em 1944 e desde de aquela época houve inovações, como os transitores e capacitores, que auxiliaram a evolução da tecnologia.

A internet veio de um processo evolutivo, também. Inicialmente Vinton Cerf e Robert Kahn criaram o TCP/IP. Logo depois dele Tim Berners criou a www.

Cliente e Servidor

O cliente é quem usa a internet para alcançar o Servidor. Este é uma entidade que é responsável por armanezar dados, informações, lugares. Após a solicitação do cliente, o servidor envia a informação de volta, através da internet e o Browser mostra essa informação.

Hospedagem

A hospedagem de um site consiste em armazenar as informações de um IP em um servidor. Além disso é necessário o seu registro.

Linguagem de programação clientes e servidor

Existem certas linguagens de programação que só podem ser usadas no lado do clientes e outra que só podemm ser usadas no lado do servidor. Do lado deste tem-se Java Script, Ruby e Php. Do outro, tem-se o Html, Css e Java Script

O Html

Significa Hyper Text Markup Language. É usado para que o navegador entenda as característica da página web. Para a passagem dessas características, são usadas as tags
